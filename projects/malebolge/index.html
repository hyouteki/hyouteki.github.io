malebolge - ML algorithms implementions from scratch

28 January 2023 •
  • python
  • ml

Machine Learning algorithms implementation from scratch in python using just numpy. This involves:

  • Unsupervised algorithms
    • Fuzzy c means
    • K means
    • Local Outlier Factor
    • Mahalanobis
    • Mean shift
  • Supervised algorithms
    • K Nearest Neighbours
    • Logistic Regression
    • Linear Regression
  • Analysis techniques
    • Fisher Discriminant Analysis
    • Independent Component Analysis
    • Linear Discrimiant Analysis
    • Principle Component Analysis
    • Silhouette Analysis
  • Helper classes
    • Cluster
    • Image Actor
    • Point

Quick Start

git clone https://github.com/hyouteki/malebolge
pip install -r requirements.txt
